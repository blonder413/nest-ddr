## [Validation](https://docs.nestjs.com/techniques/validation)

Valida las solicitudes entrantes. Nest proporciona varias formas de validar:

-   ParseIntPipe
-   ParseBoolPipe
-   ParseArrayPipe
-   ParseUUIDPipe

```sh
npm i --save class-validator class-transformer
```
